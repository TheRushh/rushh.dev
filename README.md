# rushh.dev

A modern portfolio website built with React, TypeScript, and Vite. Features smooth animations, theme switching, and interactive components.

## Features

- **Dual Theme System** - Light/dark mode with smooth transitions
- **Interactive Components** - Mouse follower spotlight, 3D card effects, animated dot grid
- **Performance Optimized** - RequestAnimationFrame, code splitting, direct DOM manipulation
- **Type Safe** - Comprehensive TypeScript throughout
- **Well Tested** - 96.4% coverage with Vitest and React Testing Library

## Quick Start

```bash
# Clone and install
git clone https://github.com/TheRushh/rushh.dev.git
cd rushh.dev
npm install

# Development
npm run dev

# Build
npm run build

# Test
npm test
```

## Scripts

| Command           | Description        |
| ----------------- | ------------------ |
| `npm run dev`     | Start dev server   |
| `npm run build`   | Production build   |
| `npm run lint`    | Check linting      |
| `npm test`        | Run tests          |
| `npm run quality` | Run quality checks |

## Deployment

Automatic deployment via GitHub Actions to [rushh.dev](https://rushh.dev) on push to `main`.
